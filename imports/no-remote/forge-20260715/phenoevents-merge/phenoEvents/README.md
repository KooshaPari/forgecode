# PhenoEvents

This repository has been **absorbed** into [`KooshaPari/Eventra`](https://github.com/KooshaPari/Eventra).

All active development, issues, and PRs should be directed there.

## What happened

The `phenoEvents` event-bus library (Sqlite outbox, DLQ, schema registry, projections)
has been preserved as a **historical snapshot** in
[`phenotype-tooling/docs/absorbed-from-PhenoEvents/`](https://github.com/KooshaPari/phenotype-tooling/tree/main/docs/absorbed-from-PhenoEvents)
and its code contributed to new crate `Eventra/rust/phenotype-event-store/`.
