# Specifications

## Scope

- Add Sladge disclosure to `README.md`.
- Add session docs for current-head governance evidence.
- Do not modify Rust crates, dependency files, deny policy, plans, tests, or
  generated artifacts.

- README contains the Sladge badge.
- Session docs name the stale branch and dirty canonical blocker.
